### 어려운/낯선 문제 해커톤
Flutter + NestJS를 활용하여 멀티플레이를 지원하는 웹/앱 테트리스를 만드는 것을 목표로 한다.

### 기능 구현 목록
**Front-end(Flutter)**
- [x] 보드 모델 구현
- [x] 테트로미노 블록 정의 및 스폰/랜덤 생성
- [x] 충돌 검사
- [x] 중력 + 고정 로직
- [x] 줄 삭제 + 점수 갱신 로직
- [ ] 렌더링(그리드 UI) + 키 입력(좌/우/하/회전)
- [ ] 게임 루프 제작 + 게임 오버 처리
- [ ] 다음 블록 큐 및 hold 옵션
- [ ] 레벨/속도, 일시정지, 소리 등 부가기능

**Back-end(NestJS)**

**유저 시스템**
- [ ] 회원가입/로그인 API (JWT 인증)
- [ ] 유저 프로필 조회/수정
- [ ] 비밀번호 암호화 (bcrypt)
- [ ] 토큰 갱신(refresh token)

**랭킹 시스템**
- [ ] 싱글 플레이 게임 결과 저장 API
- [ ] 전체 랭킹 조회 (페이지네이션)
- [ ] 개인 최고 기록 조회
- [ ] 일간/주간/전체 랭킹 분리
- [ ] 랭킹 캐싱 (Redis 옵션)

**멀티플레이 시스템**
- [ ] WebSocket 연결 관리 (Gateway)
- [ ] 방 생성/입장/퇴장
- [ ] 방 목록 조회 및 검색
- [ ] 게임 시작/종료 신호 처리
- [ ] 실시간 게임 상태 동기화 (블록 위치, 보드 상태)
- [ ] 플레이어 간 공격 로직 (줄 삭제 시 상대방에게 줄 추가)
- [ ] 게임 결과 저장 및 승패 기록
- [ ] 재접속 처리 (끊김 복구)
- [ ] 관전 모드 (옵션)

**인프라 & 부가기능**
- [ ] 데이터베이스 설계 및 마이그레이션 (TypeORM/Prisma)
- [ ] API 문서화 (Swagger)
- [ ] 에러 핸들링 및 로깅
- [ ] 환경 변수 관리 (.env)
- [ ] CORS 설정
- [ ] Rate Limiting (API 요청 제한)
- [ ] 배포 (Docker, AWS/Vercel 등)
